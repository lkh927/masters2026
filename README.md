This repository contains the code used to generate the results for Emma Knippel's master thesis,
 'When Consumers Misjudge Privacy Costs - Naïveté in Online Information Disclosure'.

 In model.py are the functions used for the following:
        
        1. Every result in Model.ipynb that regards the baseline model, the benchmark model 
        without disclosure.

        2. Every result in Model_var_mu.ipynb.

        3. Every result in Model_var_sigma.ipynb.

In model_sym.py are the functions used for the following:

        1. Every result in Model.ipynb that regards the symmetric specification of the 
        baseline model.

In model_naive.py are the functions used for the following:

        1. Every result in Model.ipynb that regards the case where naïve consumers are 
        introduced to the baseline model.
